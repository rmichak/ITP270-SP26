# ITP 270 — Programming for Cybersecurity (Spring 2026)

**Northern Virginia Community College**
**Instructor:** Randy Michak

## Course Overview

Introduction to Python programming with a cybersecurity focus. Students learn programming fundamentals — variables, control structures, functions, file I/O — then apply them to real-world security scenarios.

## Modules

| Module | Topic | Folder |
|--------|-------|--------|
| 1 | Computational Thinking & Programming Tools | `module-01/` |
| 2 | Variables & Data Types | `module-02/` |
| 3 | Strings | `module-03/` |
| 4 | Decision Control Structures (IF/ELSE/ELIF) | `module-04/` |
| 5 | Repetition Control Structures (WHILE/FOR) | `module-05/` |
| 6 | Lists, Tuples & Dictionaries | `module-06/` |
| 7 | Functions & Exceptions | `module-07/` |
| 8 | File Operations & Modules | `module-08/` |

## Setup

1. Install [Python 3.x](https://www.python.org/downloads/)
2. Install [VS Code](https://code.visualstudio.com/)
3. Install the Python extension in VS Code

Each module folder contains:
- `examples/` — Code examples from lectures
- `in-class/` — In-class assignment starter files
- `homework/` — Homework assignment starter files
